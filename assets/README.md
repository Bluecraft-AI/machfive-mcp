# Assets

This folder contains images and assets for the MachFive MCP server.

## Icon Requirements

For the MCP Registry and directory listings, add your icon as:

- **File name:** `icon.png`
- **Recommended size:** 256x256px or 512x512px
- **Format:** PNG (transparent background recommended)
- **Alternative formats:** SVG, JPEG, WebP (update `mimeType` in `server.json` if using)

The icon will be displayed on:
- Official MCP Registry (modelcontextprotocol.io/registry)
- MCP.so directory
- Other MCP server directories

## GitHub URL

Once uploaded, the icon will be accessible at:
```
https://raw.githubusercontent.com/Bluecraft-AI/machfive-mcp/main/assets/icon.png
```

This URL is already configured in `server.json` and `publish-registry.sh`.
